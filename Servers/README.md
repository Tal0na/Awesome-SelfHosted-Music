# 🎼 Servers

## Platforms that host and stream your music library

- [Navidrome](server/navidrome.md) – Lightweight platform for remote streaming of your music collection, compatible with Subsonic/Airsonic.
- [Jellyfin](server/jellyfin-server.md) – Full-featured media server (music, videos, photos), open-source, with web and mobile apps.
- [Plexamp](server/plexamp.md) – Music-focused player for Plex; requires Plex Media Server.
- [Airsonic](server/airsonic.md) – Modern fork of Subsonic, ideal for personal streaming.
- [Subsonic](server/subsonic.md) – Classic music streaming server, basis for Airsonic.
- [Funkwhale](server/funkwhale.md) – Federated server, allows sharing libraries across instances.
- [Ampache](server/ampache.md) – Web-based media server, compatible with various clients and apps.
